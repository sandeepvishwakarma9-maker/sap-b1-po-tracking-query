# SAP B1 PO Tracking Query

This repository contains a SAP Business One HANA SQL query used for tracking Purchase Orders to Payments.

---

## Features

- Purchase Order Details
- GRPO Tracking
- AP Invoice Tracking
- Outgoing Payment Tracking
- Payment Mode
- Overdue Days

---

## Database

SAP HANA

---

## Modules Used

- OPOR
- POR1
- OPDN
- PDN1
- OPCH
- PCH1
- OVPM
- VPM2

---

## Purpose

This query helps track:

PO → GRPO → AP Invoice → Payment
